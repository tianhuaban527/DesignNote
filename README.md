# 4. 字体

[找字体](https://github.com/tianhuaban527/DesignNote/blob/master/FindFont.md)

字体的重要性

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append46.jpg)

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append47.jpg)

PPT中不要用“宋体”,尽量用黑体或雅黑

## 4.1 字体类型

- 从字形上分析，字体分为有衬线和无衬线字体。

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append48.jpg)

有衬线字体代表着：高贵/优雅/艺术/复古；

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append49.jpg)

无衬线字体代表着：现代/简洁/低调；

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append50.jpg)

- 从笔画粗细上分析，字体还分为纤细型与粗壮型字体：

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append51.jpg)

纤细型字体代表着：女性/优雅/气质/清新/高端；

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append52.jpg)

粗壮型字体代表着：男性/刚劲/豪放/粗犷/霸气；

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append53.jpg)

## 4.2 字体的性格

> 中规中矩型：
> 此类字体字形规矩、结构清晰，最大的特点就是没有特点。

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append54.jpg)

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append55.jpg)

> 威武刚劲型：
> 此类字体字形犀利、笔画粗壮，一看就是黑道大哥专用字体。

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append56.jpg)

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append57.jpg)

> 高端气质型：
> 此类字体笔画细腻纤长，字形气质优雅，一看就是字体里的贵族。

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append58.jpg)

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append59.jpg)

> 霸气侧漏型：
> 中国的书法字，有的清秀细腻，有的大气磅礴。

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append60.jpg)

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append61.jpg)

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append62.jpg)

> 可爱清新型：
> 这种字体适合用于幼教型PPT中。

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append63.jpg)

![](https://raw.githubusercontent.com/tianhuaban527/DesignNote/master/PIC/append64.jpg)
